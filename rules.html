Server Rules

Make sure to follow these rules to keep the server fun and safe for everyone!

  • 🛑 No griefing or stealing.

  • 🚫 No hacking, cheating or use of unfair mods.
  • 🏗️ Don't build inappropriate structures.
  • 🌊 Stay on theme when building in public areas (aquatic/clean).
  • 📣 Use Discord for reporting and support.
